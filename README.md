# 벼락부자 주식단타단

2026 미니게임 메이커스 챌린지 아이디어를 바탕으로 제작한 브라우저용 웹게임입니다.

## 실행 방법

1. 폴더 안의 `index.html`을 더블 클릭합니다.
2. 크롬, 엣지, 파이어폭스 등 최신 브라우저에서 실행합니다.
3. 별도 서버나 설치 과정은 필요하지 않습니다.

## 포함 기능

- 클릭 노동 수익과 콤보
- 4종 자동화 업그레이드
- 4개 가상 종목과 단계별 해금
- 실시간 랜덤워크 차트
- 호재·악재 뉴스 이벤트
- 매수·매도와 평균 단가, 평가 손익
- 자동 손절·익절
- 3분 제한 세션과 결과 등급
- 모바일 반응형 UI
- 가벼운 효과음

## 파일 구성

- `index.html`: 화면 구조
- `style.css`: 디자인과 반응형 레이아웃
- `game.js`: 게임 상태, 시세, 뉴스, 거래, 업그레이드 로직


## 주가 밸런스

- 주가 갱신 주기: 2.5초
- 1회 변동 제한: 상승 +30%, 하락 -30%
- 뉴스 효과도 한 번의 갱신에서는 위 제한을 초과하지 않습니다.

- 별도 보유주식 창: 전 종목 수량, 평균가, 현재가, 평가금, 손익률 확인 및 전량 매도

- 주문 수량 빠른 선택 버튼을 +1/+5/+10 누적 방식으로 변경
- 뉴스 발생 확률을 호재 70%, 악재 30%로 조정

- 타임어택 종료 방식 제거
- 목표 자산 300,000원 달성 시 게임 종료
- 플레이 시작부터 목표 달성까지 경과 시간 기록
- 목표 달성 전까지 주가 변동과 뉴스 이벤트가 계속 발생
- 결과 화면에 목표 달성 시간과 기록 평가 표시

- 목표 자산을 1,000,000원으로 상향
- 클릭 및 자동 수익을 전반적으로 약 30% 감소
- 자산 400,000원에서 해금되는 `불장지수 2X` 추가
- 레버리지 상품은 일반 가격 변동과 뉴스 영향을 2배로 반영
- 레버리지 적용 후에도 1회 가격 변동은 ±30%로 제한
