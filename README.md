# 벼락부자 주식단타단

2026 미니게임 메이커스 챌린지 아이디어를 바탕으로 제작한 브라우저용 웹게임입니다.

## 실행 방법

1. 폴더 안의 `index.html`을 더블 클릭합니다.
2. 크롬, 엣지, 파이어폭스 등 최신 브라우저에서 실행합니다.
3. 별도 서버나 설치 과정은 필요하지 않습니다.

## 포함 기능

- 클릭 노동 수익과 콤보
- 4종 자동화 업그레이드
- 4개 가상 종목과 단계별 해금
- 실시간 랜덤워크 차트
- 호재·악재 뉴스 이벤트
- 매수·매도와 평균 단가, 평가 손익
- 자동 손절·익절
- 3분 제한 세션과 결과 등급
- 모바일 반응형 UI
- 가벼운 효과음

## 파일 구성

- `index.html`: 화면 구조
- `style.css`: 디자인과 반응형 레이아웃
- `game.js`: 게임 상태, 시세, 뉴스, 거래, 업그레이드 로직
